O código background-color define a cor de fundo do container, enquanto o background-image define uma imagem de fundo para o container.

Quando a imagem escolhida é muito grande, ela cobre todo o container mas não se redimensiona sozinha. Quando ela é muito pequena, ela se repete, cobrindo todo o container.





Ainda no background-image, podemos definir um linear-gradient para fazer um efeito gradiente.

Pode-se definir várias cores para o gradiente, mas depende do resultado desejado.
