ATENÇÃO para a sequencia dos lados que o html considera como order, quando há mais de um valor para a propriedade (cor, tipo de borda, etc.)

Exemplo 3: border-color

A ordem do HTML para mais de um valor na propriedade é:

  • 1 valor: top, bottom, left, rigth
  • 2 valores: top, bottom // left, rigth
  • 3 valores: top // left, rigth // bottom
  • 4 valores: top // rigth // bottom // left
